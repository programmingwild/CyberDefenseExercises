# Cyber Defense Practical Exercises

A collection of practical cybersecurity tasks designed to develop hands-on skills in network security, vulnerability assessment, traffic analysis, web application security, and security reporting. Each task includes documentation, reports, screenshots, scripts, and demonstrations to provide practical experience with industry-standard cybersecurity concepts and tools.

---

## Repository Overview

This repository covers a variety of cybersecurity domains including:

* Network Reconnaissance
* Firewall Configuration
* Web Application Security Testing
* Vulnerability Assessment
* Traffic Analysis
* Security Awareness Research
* Security Reporting

---

## Tasks Included

### Task 1 – Basic Network Scanning with Nmap

Perform network reconnaissance and identify active hosts, open ports, and running services using Nmap.

**Skills Gained**

* Host Discovery
* Port Scanning
* Service Enumeration
* Network Mapping

**Tools**

* Nmap
* Ping
* Traceroute

---

### Task 2 – Basic Firewall Configuration using UFW

Configure firewall rules to control network traffic and enhance system security.

**Skills Gained**

* Firewall Management
* Access Control
* Traffic Filtering
* Security Hardening

**Tools**

* UFW (Uncomplicated Firewall)
* Linux Terminal

---

### Task 3 – SQL Injection on DVWA (Low Security)

Understand and demonstrate SQL Injection vulnerabilities in a controlled DVWA environment.

**Skills Gained**

* Web Security Testing
* SQL Injection Identification
* Database Enumeration Concepts

**Tools**

* DVWA
* Web Browser
* SQL Queries

---

### Task 4 – Research Report on Common Network Security Threats

Research and document modern network security threats affecting organizations and individuals.

**Topics Covered**

* Malware
* DDoS Attacks
* Phishing
* Man-in-the-Middle Attacks
* Insider Threats

---

### Task 5 – Research Report on Social Engineering Attacks

Analyze common social engineering techniques used to manipulate users and gain unauthorized access.

**Topics Covered**

* Phishing
* Spear Phishing
* Pretexting
* Baiting
* Tailgating

---

### Task 6 – Research Report on the Importance of Patch Management

Study the role of patch management in reducing vulnerabilities and maintaining secure systems.

**Topics Covered**

* Security Updates
* Vulnerability Mitigation
* Risk Reduction
* Compliance Requirements

---

### Task 7 – Vulnerability Scanning with Nikto

Perform web server vulnerability assessments using Nikto.

**Skills Gained**

* Vulnerability Identification
* Web Server Assessment
* Security Analysis

**Tools**

* Nikto
* Kali Linux

---

### Task 8 – Capture Network Traffic with Wireshark

Capture and analyze network packets to understand communication patterns and detect anomalies.

**Skills Gained**

* Packet Analysis
* Protocol Inspection
* Network Troubleshooting
* Security Monitoring

**Tools**

* Wireshark

---

### Task 9 – Exploit a Vulnerability in a Web Application (SQL Injection)

Demonstrate the exploitation process of a known web application vulnerability in a controlled environment.

**Skills Gained**

* Vulnerability Validation
* Exploitation Concepts
* Security Testing Methodology

**Tools**

* DVWA
* Web Browser
* SQL Queries

---

### Task 10 – Write a Detailed Security Assessment Report for a Network

Conduct a complete security assessment and prepare a professional security report.

**Skills Gained**

* Risk Assessment
* Vulnerability Documentation
* Security Reporting
* Remediation Planning

**Deliverables**

* Findings
* Risk Ratings
* Recommendations
* Executive Summary
